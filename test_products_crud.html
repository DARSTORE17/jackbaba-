🔧 API Endpoints

GET /seller/products - List all products (with pagination)
POST /seller/products - Create new product
GET /seller/products/{id} - Get product details
PUT /seller/products/{id} - Update product
DELETE /seller/products/{id} - Delete product
PATCH /seller/products/{id}/toggle-advertised - Toggle advertised status

📊 Database Tables

Products Table:
id, category_id, name, slug, old_price, new_price, discount, rate, stock, thumbnail, is_advertised, created_at, updated_at
Product Media Table:
id, product_id, type (image/video), file_path, is_primary, created_at, updated_at
Product Descriptions Table:
id, product_id, description, specifications, details, created_at, updated_at

🚀 How to Test

  1. Access the Products Page:
    Navigate to: /seller/products
  2. Add a New Product:
    1. Click "Add New Product" button
    2. Fill in the form (Name, Category, Price, Stock required)
    3. Optionally upload a thumbnail
    4. Click "Save Product"
    5. See SweetAlert success notification
  3. Edit a Product:
    1. Click the Edit button (pencil icon)
    2. Modify any fields
    3. Click "Update Product"
    4. See success notification
  4. View Product Details:
    1. Click the View button (eye icon)
    2. See detailed product information in modal
  5. Delete a Product:
    1. Click the Delete button (trash icon)
    2. Confirm deletion in SweetAlert dialog
    3. See success notification
  6. Toggle Advertised Status:
    1. Click the toggle switch in the Status column
    2. See instant update with notification
  7. Search & Filter:
    1. Use search box to find products by name
    2. Use category dropdown to filter by category
    3. Click "Filter" or "Reset" buttons
  8. Pagination:
    1. Navigate through pages (10 items per page)
    2. Click page numbers or Next/Previous buttons
